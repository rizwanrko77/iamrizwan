# Mohd Rizwan - Portfolio

Personal portfolio and professional website for Mohd Rizwan.

## Tech Stack
- Next.js (App Router)
- React
- TypeScript
- Plain CSS (Custom Design System)

## Local Development

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment
This project is configured for static export (`output: 'export'`) and is optimized for deployment on Cloudflare Pages.
